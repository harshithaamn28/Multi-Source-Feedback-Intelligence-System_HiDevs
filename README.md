#Multiple Source Feedback Intelligence System 
## Overview 
This project is a Python-based Feedback Intelligence System that analyzes the feedback from multiple sources.It performs sentiment analysis to categorize feedback into issue based such as bugs,performance issues ,feature request.

##  Features
Sentiment Analysis (Positive, Negative, Neutral)
Issue Based Categorization (Bug, Performance, Feature Request)
Interactive Dashboard using Streamlit
CSV-based feedback data processing
FastAPI backend for handling feedback requests
Real-time filtering and visualization

##  Tech Stack
Python
Streamlit
FastAPI
TextBlob

 ##  Output
Sentiment distribution charts
Issue category analysis
sortable feedback table
Summary metrics
